# Smart School Attendance & Parent Alert System

A modern web-based attendance management system built with Google Apps Script.

## Features
- Teacher Login System
- Class-wise Student Attendance
- Real-time Summary Cards + Progress Bar
- Automatic Parent Email Alerts for Absences
- Beautiful Modern UI
- CSV Download for Attendance & Reports
- Search & Mark All Present functionality

## How to Use
1. Open the deployed web app link
2. Login with teacher credentials
3. Select class → Load Students → Mark Attendance
4. View Reports & Download CSV

## Tech Stack
- Google Apps Script
- HTML5 + CSS3 + JavaScript
- Google Sheets (as database)

## Setup
- Google Sheet with sheets: `Teachers`, `Classes`, `Students`, `Attendance`
- Deploy as Web App
